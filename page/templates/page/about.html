About Ecommerce Website

Author
Brady Vitrano

Purpose

To have an ecommerce type website for test automation of cloud products.

Features

Ecommerce website has limited functionality due to its limited requirements.

• Catalog

Browse a list of products by category.

  • The catalog page /catalog/ lists all the Products which have been flagged as featured.
  • The left hand navigation are all active categories by title.
  • You can click on a category to view the category page which lists all products by category.
  • The category page /catalog/1/ has only active products.
  • The category page /catalog/1/ also has a specials link to view website specials.

• Products

Let's sell some products!

  • Some Properties (see api for the rest):
    • special - used to mark a product as on special
    • status - allowed values (active, inactive, archived, deleted)
    • small_image - image url. This should be 320x150px
    • large_image - image url. This should be 800x300px

• Specials

The website allows for marking products as on special via the Product special flag.

  • The specials page /specials/ works conditionally.

    This will show either all products on special OR by category based on COOKIE['PRODUCTS']

    1. If the cookie PRODUCTS is not present. It will show all Products with the special flag
    2. If the cookie PRODUCTS IS present. It will show only products in that category with the special flag.

    To show products on special by category, add the PRODUCTS cookie and set it to a Category id

• Shopping Cart

The shopping cart has limited functionality. It should not be cached by any services.

  • You can add products to the shopping cart by clicking Add to Cart
  • The shopping cart /cart/ only supports adding and removing products. However, that is all!

• Blog

The blog serves as an example section of a site that should be cached as much as possible.

  • Some Properties (see api for the rest):
    • status - allowed values (active, inactive, archived, deleted)
    • img - image url. This should be 900x300px
