API Documentation

API allows for seamless integration with ecommerce website for automation.

This API only responds with application/json.

  • Product Categories - Requirement for products. One to Many relationship with products.

    • GET /v1/categories/ - Fetch all categories
    • POST /v1/categories/ - Add a category
    • GET /v1/categories/:id - Fetch a specific category
    • PUT /v1/categories/:id - Edit a category
    • DELETE /v1/categories/:id - Delete a category

  • Products - Items for purchase on website

    • GET /v1/products/ - Fetch all products
    • POST /v1/products/ - Add a product
    • GET /v1/products/:id/ - Fetch a specific product
    • PUT /v1/products/:id/ - Edit a product
    • DELETE /v1/products/:id/ - Delete a product

  • Blog - Articles of the website.

    • GET /v1/articles/ - Fetch all articles
    • POST /v1/articles/ - Add an article
    • GET /v1/articles/:id/ - Fetch a specific article
    • PUT /v1/articles/:id - Edit an article
    • DELETE /v1/articles/:id - Delete an article
